# --- EPUB validation ---
# Wraps the W3C `epubcheck` jar so `./binder check epub --scope epubcheck`
# can validate built EPUBs without a separate installation step. The
# wrapper bundles the jar but requires a Java runtime (JRE 8+) on the
# host — install via your OS package manager (`brew install temurin`,
# `apt install default-jre`, etc.) or the `actions/setup-java` action
# in CI. When the package is absent, the binder check falls back to
# the `epubcheck` command on PATH and surfaces a clear install hint if
# neither is found, so a missing JRE never produces a silent skip.
