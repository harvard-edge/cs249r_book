Companion Textbook#

Machine Learning Systems#

mlsysbook.ai by Prof. Vijay Janapa Reddi (Harvard University)

TinyTorch began as hands-on labs for this textbook. While TinyTorch can be used standalone, the ML Systems book provides the theoretical depth and production context behind every module you build.

What it teaches: Systems engineering for production ML‚Äîmemory hierarchies, performance optimization, deployment strategies, and the engineering decisions behind modern ML frameworks.

How it connects to TinyTorch:

  • TinyTorch modules directly implement concepts from the book‚Äôs chapters

  • The book explains why PyTorch, TensorFlow, and JAX make certain design decisions

  • Together, they provide both hands-on implementation and theoretical understanding

When to use it: Read in parallel with TinyTorch. When you implement Module 05 (Autograd), read the book‚Äôs chapter on automatic differentiation to understand the systems engineering behind your code.
